# 会议助手

 这个是为了实现一个WebAPP，主要想实现的是方便用户对相关会议进行方便记录与管理。
 
## 功能

### 管理员

1. 会议室设定
2. 账号分配

······

### 用户

1. 新建会议
2. 会议室状态
3. 会议纪要
4. 我相关的
5. 信息设置

······

## 待增加的功能

1. 新增会议时需要时间约束，按小时、会议室分开会议室占用状态，点击翻页（下一周），点击空白格可以选取对应时间，点击有色格展示占用人信息
2. 我的会议，两种：已完成的会议、待进行的会议(ing···)
3. 会议详情，功能
